
# Employee Payroll Tracker


The Employee Payroll Tracker is a web-based application that helps track employee details such as first name, last name, and salary. It allows users to add employees, view the list of employees, calculate the average salary, and select a random employee for special mentions or awards.


## Features

- Add employee details including first name, last name, and salary.

- Display employee information in a table format.

- Calculate and display the average salary of all employees.

- Select and display a random employee from the list.


## Technologies Used

- HTML

- CSS

- JavaScript

## Usage

1. Open the application by opening index.html in your web browser.

2. Click on the "Add Employees" button to start adding employee details.

3. Follow the prompts to enter the first name, last name, and salary for each employee.

4. Confirm if you want to add more employees or finish adding.

5. View the updated employee list in the table.

6. The average salary will be calculated and displayed in the console.

7. A random employee will be selected and displayed in the console for special mention.
